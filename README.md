## 📖 Sobre o Projeto

O **MyIoga** é uma plataforma online desenvolvida para disponibilizar aulas de yoga exclusivas para alunos. O projeto tem como objetivo oferecer um ambiente organizado e acessível para acompanhamento das práticas, promovendo bem-estar, constância e evolução na jornada do yoga.

O sistema foi criado para centralizar:
* 📹 **Aulas gravadas**
* 📅 **Organização das práticas**
* 📚 **Conteúdos complementares**
* 🔐 **Área exclusiva para alunos**

> A proposta é transformar a experiência da prática em algo contínuo, acessível e personalizado.

---

## ✨ Funcionalidades

- [x] **Sistema de login** para alunos.
- [x] **Área restrita** com acesso às aulas.
- [x] **Organização de vídeos** por categoria ou nível.
- [x] **Estrutura preparada** para expansão futura.

---

## 🛠️ Tecnologias Utilizadas

A stack principal escolhida para garantir estabilidade e escalabilidade:

* **Frontend:** Angular v20 & TypeScript
* **Ambiente de Execução:** Node.js
* **Estilização & Estrutura:** HTML5 e CSS3

---

## 🎯 Objetivo

Criar uma plataforma própria para disponibilização de aulas online, fortalecendo a conexão com os alunos e oferecendo autonomia digital para o ensino de yoga.
